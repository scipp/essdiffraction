ess.dream.diagnostics.FlatVoxelViewer#

class ess.dream.diagnostics.FlatVoxelViewer(data, *, rasterized=True, **kwargs)[source]#

Interactive 2D plot of all detector voxels.

See DREAM Detector Diagnostics for explanations and an example.

__init__(data, *, rasterized=True, **kwargs)[source]#

Create a new viewer.

Parameters:

  • data (Mapping[str, DataArray]) – Histogrammed data, one entry per bank.

  • rasterized (bool, default: True) – If True, the figure is rasterized which improves rendering speed but reduces resolution.

  • **kwargs (Any) – Additional arguments passed to the plotting function.
